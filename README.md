# AOUU

## 简介

AOUU 是一个简单的**本地**工具，而非游戏mod，用于在《黑夜君临》中，当“执行者”释放大招时，自动播放 Animals 的高潮片段，用于增强游戏演出效果。

## 功能

* 监听指定按键（对应大招触发）
* 播放本地音频文件（支持 mp3）
* 可自定义音频路径
* 可配置触发按键
* 检测大招就绪时才会触发

* ## 使用方法

1. 运行程序

2. 完成基础配置后，按以下流程使用：
   1、配置好截图按键和技能按键
   2、进入到训练场，**在有大招的情况下**按照提示完成初始化框选
   3、享受你的游戏

4. 在游戏中触发技能时，将自动播放音频
