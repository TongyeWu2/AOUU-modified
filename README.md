# AOUU

## 简介

AOUU 是一个简单的**本地**工具，而非游戏mod，用于在《黑夜君临》中，当“执行者”释放大招时，自动播放 Animals 的高潮片段，用于增强游戏演出效果。

## 功能

* 监听指定按键（对应大招触发）
* 播放本地音频文件（支持 mp3）
* 可自定义音频路径
* 可配置触发按键
* 检测大招就绪时才会触发

## 使用方法


1. 解压后运行 `AOUU.exe`
2. 配置好截图按键和技能按键
3. 进入训练场，在有大招的情况下，按照提示完成初始化框选
4. 正常进入游戏，触发技能后将自动播放音频
